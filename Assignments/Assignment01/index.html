Software Development

Software development is the process of computer programming, documenting, testing and bug fixing involved in creating and maintaining applications and frameworks involved in a software release life cycle and resulting in a software product.

Web Development

Web development, a broad term for the work involved in developing a web site for the internet or an intranet. It can range from developing the simplest static single page of plain text to the most complex web-based internet applications, electronic businesses and social network services.

Mobile Applications

Mobile application development is the set of processes and procedures involved in writing software for small, wireless computing devices such as smartphones or tablets. Mobile application development has its roots in more traditional software development.
